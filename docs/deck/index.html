The Problem

  • 42% of code committed today is AI-assisted (GitHub, 2025)
  • EU AI Act Article 50 mandates disclosure of AI-generated content
  • Enforcement begins August 2, 2026. Fines reach 3% of global revenue
  • git blame shows who committed. It does NOT show who or what wrote the code

The Gap

C2PA exists for images and video.
Nothing exists for source code.

No standard records which model wrote which function, who approved it, or whether a human was involved at all.

Three Authorship Scenarios

Human Only

Developer writes every line. No AI involvement. No approval gate needed.

kind: humanhumanShare: 1.0

AI-Assisted

Human and AI collaborate. Human edits AI output. ECDSA approval signature required.

kind: ai-assistedhumanShare: 0.0 to 1.0

AI Autonomous

AI generates code independently. Human only approves. Model identity cryptographically recorded.

kind: ai-autonomoushumanShare: null

Built with IBM Bob

Custom Mode: Provenance Officer

  • Reads AGENTS.md for risk classification
  • Gates HIGH-risk commits on human approval
  • Computes SHA-256 content hashes for every field
  • Invokes pedigree-mcp via MCP protocol
  • Refuses to push unsigned commits
